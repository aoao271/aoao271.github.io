# 微分の基本

## 問題：次の関数を微分せよ

1. $y = -\sin \theta$

2. $y = \log x$

3. $y = 5x^2$

4. $y = 3^{2x}$

5. $y = 4x^{0.25}$

!!!+ info "ヒント"
    - 第1問：三角関数の微分
    - 第2問：対数関数の微分
    - 第3問：べき関数の微分
    - 第4問：指数関数の微分（対数を使う）
    - 第5問：べき関数の微分（分数指数）
