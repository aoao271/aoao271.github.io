# 部分積分

## 問題：次の式を計算せよ

1. $\displaystyle \int_{}^{} \log x dx$

2. $\displaystyle \int_{}^{} x^2\sin x dx$

3. $\displaystyle \int_{}^{} \sin x\cos x dx$

4. $\displaystyle \int_{1}^{6} x\log x dx$

5. $\displaystyle \int_{0}^{4} (2 - x)^2 e^x dx$

!!!+ info "ヒント"
    - 第1問：$u = \log x$, $dv = dx$
    - 第2問：多項式と三角関数の積
    - 第3問：$\sin 2x$への変形も可
    - 第4問：定積分での部分積分
    - 第5問：指数関数と多項式の積
