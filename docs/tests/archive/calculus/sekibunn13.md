# 微分総合

## 問題：次の関数を微分せよ

1. $y = x$

2. $y = \dfrac{1}{\sqrt{x^3}}$

3. $y = \cos x + 2\tan x$

4. $y = 5\log x$

5. $y = 7^x$

6. $y = e^{2x}\log x$

7. $y = 3^{3x - 1}$

8. $y = x^{e^x}$

9. $y = \dfrac{x^2 - 6x -7}{x^2 - 1}$

10. $y = \dfrac{1}{5}e^x\sin x\cos x - \dfrac{1}{5}e^x\cos 2x$

!!! warning "注意"
    問題10は+,-の記号を使わずに解答すること

!!!+ info "微分の公式"
    - 積の微分：$(uv)' = u'v + uv'$
    - 商の微分：$(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$
    - 合成関数：$y = f(g(x)) \Rightarrow y' = f'(g(x))g'(x)$
