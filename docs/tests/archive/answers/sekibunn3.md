# 不定積分の基本 - 解答

!!! warning "注意"
    Cは積分定数

1. $e^x + C$

2. $\dfrac{5^x}{\log 5} + C$

3. $-\cos x + C$

4. $-\dfrac{1}{\tan x} + C$

5. $3\log |x| + C$

6. $2\log |\cos x| + C$

7. $x\log x - x + C$

8. $\dfrac{1}{18}x^2 + C$

9. $4x^{\frac{1}{4}} + C$

    !!! note "解説"
        $\begin{align}
        \int \dfrac{1}{\sqrt[4]{x^3}} dx & = \int x^{-\frac{3}{4}} dx \\
        & = 4x^{\frac{1}{4}} + C
        \end{align}$

10. $x^3 + C$
