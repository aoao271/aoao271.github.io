問題：次の関数を微分せよ

    ※Cは積分定数

  • (1)の答え：\( e^x + C \)
  • (2)の答え：\( \dfrac{5^x}{\log 5} + C \)

  • (3)の答え：\( -\cos x + C \)

  • (4)の答え：\( -\dfrac{1}{\tan x} + C \)

  • (5)の答え：\( 3\log |x| \) + C
  • (6)の答え：\( 2\log |\cos x| + C \)

  • (7)の答え：\( x\log x - x + C \)

  • (8)の答え：\( \dfrac{1}{18}x^2 + C \)

  • (9)の答え：\( 4x^{ \frac{1}{4} } + C \)

    解説：

    \( \hspace{35px} \begin{eqnarray} \displaystyle \int_{}^{} \dfrac{1}{ \sqrt[4]{x^3} } dx & = & \displaystyle \int_{}^{} x^{ -\frac{3}{4} } dx \\ & = & 4x^{ \frac{1}{4} } + C \end{eqnarray} \)

  • (10)の答え：\( x^3 + C \)
